
The folder srcMex contains the necessary files for compiling MEX-files used in MLToolbox.
It's necessary  to configure Matlab for compiling MEX-files, please refer to your Matlab
manual to know how to get that.
If everything is configured, it's only necessary to execute the MAKE script.

This files have been written to be compatible with UNIX an WINDOWS, and have been compiled
sucesfully in Windows and Solaris

For compiling under Windows with GNU Compiler (cygwin or mingw), you can find useful this
URL: http://gnumex.sourceforge.net/
